# Apollo Clicker

A lightweight Cookie Clicker-style browser game starring Apollo the parrot.

## Play locally

```bash
python -m http.server 4173
```

Then open <http://localhost:4173> in your browser.

## How to play

- Click Apollo to earn seeds.
- Buy upgrades to increase seeds per click and passive seeds per second.
- Progress is auto-saved in your browser.
- Use **Reset save** to start over.
